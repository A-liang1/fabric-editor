# Fabric Editor 设计架构文档

## 一、项目概述

Fabric Editor 是一个基于 Fabric.js + React + TypeScript 的画布编辑器，支持基础图形操作和 Undo/Redo 功能。

### 功能特性

- 🎨 基础图形绘制（矩形、圆形、文本）
- 🖱️ 对象操作（移动、缩放、删除）
- ↩️ Undo/Redo 历史记录
- 🎛️ 实时属性编辑（颜色、透明度等）
- ⌨️ 快捷键支持（Ctrl+Z/Y、Delete）

### 技术栈

- **Canvas 引擎**: Fabric.js 7.x
- **前端框架**: React 19 + TypeScript
- **构建工具**: Vite 8.x
- **状态管理**: React Context

## 三、核心设计模式

### 3.1 Command 命令模式

```
┌─────────────────────────────────────────┐
│           Command Interface             │
│  ┌─────────────────────────────────┐    │
│  │  execute(): void                │    │
│  │  undo(): void                   │    │
│  │  redo(): void                   │    │
│  └─────────────────────────────────┘    │
└──────────────────┬──────────────────────┘
                   │
    ┌──────────────┼──────────────┬──────────────┐
    ▼              ▼              ▼              ▼
┌────────┐   ┌────────┐    ┌──────────┐   ┌──────────┐
│  Add   │   │ Delete │    │   Move   │   │  Scale   │
│Object  │   │Object  │    │ Object   │   │ Object   │
│Command │   │Command │    │ Command  │   │ Command  │
├────────┤   ├────────┤    ├──────────┤   ├──────────┤
│execute:│   │execute:│    │execute:  │   │execute:  │
│  add() │   │remove()│    │  toX/Y   │   │ toScale  │
│undo:   │   │undo:   │    │undo:     │   │undo:     │
│ remove │   │  add() │    │ fromX/Y  │   │ fromScale│
│redo:   │   │redo:   │    │redo:     │   │redo:     │
│  add() │   │remove()│    │  toX/Y   │   │ toScale  │
└────────┘   └────────┘    └──────────┘   └──────────┘
```

### 3.2 Undo/Redo 双向栈

```
初始状态：
  undoStack: []      redoStack: []

执行操作 A (Add Rect)：
  undoStack: [A]     redoStack: []

执行操作 B (Move)：
  undoStack: [A, B]  redoStack: []

撤销一次 (Undo)：
  B.undo()           ← 恢复位置
  undoStack: [A]     redoStack: [B]

撤销两次 (Undo)：
  A.undo()           ← 删除矩形
  undoStack: []      redoStack: [A, B]

重做一次 (Redo)：
  A.redo()           ← 恢复矩形
  undoStack: [A]     redoStack: [B]

执行新操作 C (修改颜色)：
  C.execute()
  undoStack: [A, C]  redoStack: []   ← B 被丢弃！
```

## 六、总结

本项目采用 **Command 模式** 实现 Undo/Redo，通过 **React Context** 管理全局状态，**Fabric.js** 提供底层画布能力。架构清晰，职责分明，易于扩展。

**核心亮点**：

1. 命令模式封装所有操作，天然支持撤销重做
2. 双向栈管理历史记录，内存可控
3. Context 状态管理，组件通信简单
4. TypeScript 类型安全，维护成本低
